# Example Samples

Example compositions generated with Midi Music Generator (Zeamar's Fork v1.1). Each track includes both a `.mid` (MIDI) file and a `.wav` (rendered audio) file.

| Track | Model | Soundfont (WAV) |
|---|---|---|
| Flamenco | Gemini 2.5 Flash | FluidR3_GM |
| Smooth Jazz | Gemini 2.5 Flash | FluidR3_GM |
| Indie Rock | Gemini 2.5 Pro | FluidR3_GM |
| RPG Game Village Theme | Gemini 2.5 Pro | FluidR3_GM |

WAV files were rendered from the MIDI output using FluidSynth with the FluidR3_GM soundfont at gain 0.6.
